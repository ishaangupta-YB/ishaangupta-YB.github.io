# ishaangupta.tech

Personal portfolio website built with Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, and Magic UI. Deployed to GitHub Pages at [ishaangupta.tech](https://ishaangupta.tech).

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Static output is generated in the `out/` directory and deployed via GitHub Actions.
